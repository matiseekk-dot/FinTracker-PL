# FinTrack PRO 🇵🇱

Zaawansowana aplikacja do śledzenia finansów osobistych. Zbudowana w React + Vite.

## Funkcje

- **Dashboard** — przegląd finansów, wynik zdrowia finansowego, prognozy
- **Transakcje** — dodawanie, edycja, szablony szybkiego dodawania, wielowalutowość
- **Płatności** — śledzenie rachunków, subskrypcji, kredytów z checkboxami
- **Cele** — oszczędnościowe, limity kategorii, moduł wakacyjny
- **Analityka** — wykresy, rankingi, porównania miesięcy, trend kategorii
- **Konta** — zarządzanie kontami bankowymi i inwestycyjnymi

## Instalacja

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Technologie

- React 18
- Vite 5
- Recharts (wykresy)
- XLSX (eksport/import)
- Lucide React (ikony)
- localStorage (przechowywanie danych)

## Dane

Aplikacja przechowuje dane lokalnie w przeglądarce (localStorage).
Eksport/import danych dostępny w ustawieniach jako plik .xlsx lub JSON.
